# Conversor de Bases Numéricas
    Conversor de bases numéricas executado em console. Converte um número em determinada base para a seguinte base escolhida. 

### **As bases disponíveis para conversão são:** 
- Binária
- Octal 
- Decimal 
- Hexadecimal
